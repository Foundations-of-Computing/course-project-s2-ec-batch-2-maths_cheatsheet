LAPLACE TRANSFORMS

Comprehensive list of Laplace Transform equations and properties:

Basic Laplace Transforms

  • ℒ{1} = 1/s
  • ℒ{t} = 1/s²
  • ℒ{e^(at)} = 1/(s - a)
  • ℒ{sin(at)} = a/(s² + a²)
  • ℒ{cos(at)} = s/(s² + a²)

Properties of Laplace Transforms

  • Linearity Property: ℒ{af(t) + bg(t)} = aℒ{f(t)} + bℒ{g(t)}
  • Time Shifting: ℒ{e^(at)f(t)} = F(s - a)
  • Scaling in the s-Domain: ℒ{f(at)} = F(s/a)

Inverse Laplace Transforms

  • ℒ⁻¹{1/s} = 1
  • ℒ⁻¹{1/s²} = t
  • ℒ⁻¹{1/(s - a)} = e^(at)
  • ℒ⁻¹{a/(s² + a²)} = sin(at)
  • ℒ⁻¹{s/(s² + a²)} = cos(at)

Special Laplace Transforms

  • ℒ{tⁿ} = n!/sⁿ⁺¹
  • ℒ{sin²(at)} = a/(s² + 2a²)
  • ℒ{cos²(at)} = s/(s² + 2a²)
